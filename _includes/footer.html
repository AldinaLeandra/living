Living Wage Kenya calculator lets employers calculate if they are paying their employees a living wage.

The tool was originally created by OpenUp, formerly Code for South Africa, to highlight the low wages earned by domestic workers.

This site was produced in partnership with Code for Africa’s iLAB data journalism programme, with support from Deutsche Welle Akademie. All content is released under a Creative Commons 4 Attribution license. Reuse it to help empower your own community. The code is available on GitHub and data is available on ​openAFRICA​.
